# Projekt-FSST
per RFID zugriff auf Fenster mit Hilfe von Gestensteuerung

Ziel dieses Projekts ist die Entwicklung eines Zugangssystems auf Basis von RFID-Technologie, bei dem autorisierte Nutzer über eine Datenbank verifiziert werden. Nach erfolgreicher Authentifizierung wird eine Gestensteuerung mittels Kamera aktiviert, mit der ein Fenster (simuliert durch LEDs) gesteuert werden kann.

Die Umsetzung erfolgt auf einem Raspberry Pi 5 und kombiniert Hardware (RFID-Reader, Kamera, LEDs) mit Softwarekomponenten wie Datenbankanbindung und Bildverarbeitung.
